## 一、项目概述

### 1.1 项目背景

停云山居是一家位于青岛崂山的文化空间，集禅修、茶艺、美食、住宿于一体。本次开发微信小程序，旨在为散客提供便捷的扫码点单和付费体验，同时展示山居文化、会员权益和活动信息。

### 1.2 核心目标

| 目标 | 说明 |
|------|------|
| 散客扫码点单 | 客人入座后扫桌上二维码 → 浏览菜单 → 加入购物车 → 微信支付 → 后厨接单 |
| 山居文化展示 | 通过精美页面展示停云山居的环境、理念、特色服务 |
| 会员体系 | 微信手机号一键注册成为会员，查看权益和等级信息 |
| 活动管理 | 展示当前/往期活动，支持在线报名参加 |

### 1.3 设计风格

- **风格定位**：典雅、大气、中国风
- **配色方案**（你已在 HTML 原型中确认）：

| 颜色 | 色值 | 用途 |
|------|------|------|
| 山居绿（主色） | #4F6F52 | 按钮、导航选中态、品牌色 |
| 雾灰（辅助色） | #7C8A8C | 次要文字、图标 |
| 金色（点缀色） | #C9A962 | 装饰线、会员标识、特殊强调 |
| 宣纸白（底色） | #F8F6F1 | 页面背景，营造宣纸质感 |
| 暖米色 | #E8E4DC | 卡片背景、图片占位 |
| 墨黑 | #1A1A1A | 主要文字 |
| 墨灰 | #4A4A4A | 次要文字 |

- **字体**：衬线体为主（如 Noto Serif SC），搭配系统无衬线体用于数据展示
- **整体氛围**：山水画渐变背景、毛玻璃卡片、金色装饰线、留白排版

## 二、用户角色

| 角色 | 说明 | 核心需求 |
|------|------|---------|
| **散客（游客）** | 到店用餐的客人，无需注册即可点单 | 扫码→点菜→支付→查看订单 |
| **注册会员** | 已用微信手机号注册的客人 | 查看会员权益和等级、报名活动、查看历史订单 |

## 三、功能模块详解

### 模块 A：首页（Home）

**页面位置**：小程序底部Tab第1项

**包含内容**：

| 区域 | 功能 | 优先级 |
|------|------|--------|
| 开屏动画 | 停云山居品牌展示，约2.5秒自动消失 | P0 |
| 轮播图 | 3-5张图片自动轮播，展示山居环境/活动/美食 | P0 |
| 扫码点单入口 | 突出展示的快捷入口，点击进入点单页 | P0 |
| 山居介绍卡片 | 展示山居简介、特色标签、"了解详情"按钮 | P0 |
| 最近活动列表 | 展示2-3个近期活动，点击进入详情 | P0 |
| 底部Tab导航 | 首页、点单、订单、我的 四个Tab | P0 |

**跳转关系**：
- 扫码点单 → 点单页
- 山居"了解详情" → 山居介绍详情页
- 活动卡片 → 活动详情页

**当前状态**：✅ 基础功能已实现，样式已确认

---

### 模块 B：扫码点单（Menu）

**页面位置**：小程序底部Tab第2项 / 扫桌上二维码直达

**扫码点单完整流程**：

```
客人扫桌上二维码
        ↓
解析二维码中的桌号（如 A01）
        ↓
进入点单页面，顶部显示"桌号 A01"
        ↓
浏览菜品 → 加入购物车 → 去结算 → 确认订单 → 微信支付 → 下单成功
```

**菜品分类**：

| 分类 | 包含菜品 |
|------|---------|
| 套餐 | 下午茶套餐(38元)、林鸡野趣套餐(168元)、泉鱼隐逸套餐(198元)、双鲜套餐(318元) |
| 凉菜 | 黄瓜拌(18元) |
| 热菜 | 干煸崂山参(48元) |
| 主食 | 海鲜水饺(38元) |
| 饮品 | 桂花酒(28元) |

**购物车功能**：添加/减少数量、展开详情、实时合计、去结算

**支付流程**：确认订单 → 调起微信支付（已有商户号）→ 支付成功跳转详情页

**当前状态**：✅ 前端 UI 已实现，支付功能待接入

---

### 模块 C：订单管理（Orders）

**页面位置**：小程序底部Tab第3项

**功能**：订单列表（按时间倒序）、状态筛选（全部/待处理/进行中/已完成）、订单详情、联系客服、取消订单

**订单状态**：待处理 → 进行中 → 已完成，待处理可取消

**当前状态**：✅ 基础功能已实现

### 模块 D：山居介绍（About）

**入口**：首页山居介绍卡片 → "了解详情"

**内容**：顶部氛围图、关于我们、山居特色（禅修/茶艺/素斋/住宿）、环境图片、联系方式

**当前状态**：✅ 基础结构已实现，图文内容可进一步丰富

---

### 模块 E：会员中心（Member）

**入口**：个人中心 → "成为停云会员"卡片

**功能**：

| 功能 | 优先级 | 当前状态 |
|------|--------|---------|
| 会员介绍页（展示权益和等级） | P0 | ✅ 已有基础 |
| 微信手机号一键注册 | P1 | ❌ 需开发 |
| 会员信息展示（等级、积分等） | P1 | ❌ 需开发 |

**注册流程**：进入会员页 → 点击"立即注册" → 微信手机号授权 → 自动注册成功

**会员等级**（初步建议，以实际运营需求为准）：

| 等级 | 条件 | 权益 |
|------|------|------|
| 山客 | 注册即达 | 9.5折 |
| 云客 | 累计消费满1000元 | 9折、生日礼遇 |
| 停云 | 累计消费满3000元 | 8.5折、生日礼遇、优先预订 |
