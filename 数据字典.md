# 停云山居小程序数据字典 v1.0

> 制定日期：2026-06-01
> 当前用途：统一前端 mock 数据、本地存储和未来云数据库字段。
> 说明：本文件定义“业务字段”；数据库集合、索引、权限和并发策略将在接入云服务时另行补充。

## 一、通用约定

| 约定 | 说明 |
|------|------|
| ID | 业务订单使用统一可读编号；基础资料使用稳定字符串 |
| 金额 | 单位统一为人民币元，类型为 `number` |
| 时间 | 存储为 ISO 8601 字符串，例如 `2026-06-15T14:00:00+08:00` |
| 手机号 | 存储为字符串，避免数值转换 |
| 可选字段 | 无值时使用空字符串、空数组或 `null`，不使用中文占位文字 |
| 状态值 | 使用稳定的英文枚举值，页面根据枚举值展示中文 |

### 1.1 业务编号规则

格式：

```text
业务前缀 + YYYYMMDDHHmmss + 4位随机码
```

示例：

| 业务 | 前缀 | 示例 |
|------|------|------|
| 点餐订单 | `TYMEAL` | `TYMEAL20260603153908A7K2` |
| 用餐预订 | `TYDINING` | `TYDINING20260603153908B3Q9` |
| 住宿预订 | `TYROOM` | `TYROOM20260603153908C6M1` |
| 活动报名 | `TYACTIVITY` | `TYACTIVITY20260603153908D8P4` |
